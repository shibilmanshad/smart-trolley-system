# Smart Trolley System (YOLO + Barcode + Flask)

A Smart Trolley System that uses:
- YOLOv8 for AI-based product detection
- Barcode scanning for accurate billing
- Flask web app for real-time bill preview
- CSV-based product database

## Features
- Barcode-based billing for all products
- AI-based product category detection
- Live bill preview
- Budget input & warning
- Beep sound on scan

## How to Run
```bash
pip install -r requirements.txt
python app.py
